```editorconfig
1、这个配置相当于zlm流媒体服务器的账号密码，是要和流媒体服务器配置的信息一致
而且流媒体服务器的配置信息中还有IP白名单，也需要加上当前IP
media:
    id: 035c73f7bb6b
    secret: l5DPpqy3YbPE5oaJCL2AxOBOuAVwEvlG


2、SIP是填服务端地址，注意区分服务端和流媒体服务器端的区别，如果两者不在同一台服务器中就要区分好

```
